=============

Created using Google Antigravity

Web app to allow users to sign in using their Google account and then search for videos on YouTube.

Uses the following services:
- AuthService: Handles user authentication and Google OAuth2 flow
- YouTubeService: Handles YouTube API calls
- GatewayService: Handles API gateway functionality
- UI: Handles user interface functionality

MVP lets user sign in with Google and list their liked videos on YouTube (First 50 videos).
